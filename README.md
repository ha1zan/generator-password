Генератор паролей
1. Описание проекта
Данный проект представляет собой программу на языке Python для генерации случайных паролей.
2. Функциональность
Программа позволяет:
генерировать случайный пароль
задавать длину пароля
включать или отключать цифры
включать или отключать специальные символы
3. Как использовать
Скачать репозиторий
Открыть проект
Запустить файл:
python main.py
4. Цель проекта
Проект выполнен в учебных целях:
закрепление базовых знаний Python
изучение генерации случайных значений
практика написания программ
5. Примечание
Код может быть доработан и улучшен.
